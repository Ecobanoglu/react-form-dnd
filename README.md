# React Drag Drop Form 

This project was developed as an example for the development of another project. 

This project can be useful for creating automated items, survey module with react-hook-form. 

## Stacks
- React (with Vite)
- Typescript
- React Hook Form
- React Beautiful DND
- Radix UI
- Tailwind CSS
- ZOD
